Project: Customer Segmentation for Arvato Financial Services¶

In this project, I will apply unsupervised learning techniques to identify a certain set of customer within a market that form the core customer base for a mail-order sales company in Germany. When creating customer segmentation, the segments can then be used to direct marketing campaigns towards audiences that will have the highest expected rate of returns and one can know which part of the general population are more likely to be a customer and which part are not. The data that I will use has been provided by Udacity partners at Bertelsmann Arvato Analytics, and represents a real-life data science task. The Data has some restrictions of use from the company, so I decided to work on the Udacity workspace where the data is provided and ready to use.

def clean_data(df):
    """
    Perform feature trimming, re-encoding, and engineering for demographics
    data
    
    INPUT: Demographics DataFrame
    OUTPUT: Trimmed and cleaned demographics DataFrame
    """
    
    for x in range(len(feat_info)):
        missing_or_unknown = feat_info.iloc[x]['missing_or_unknown']
        missing_or_unknown = missing_or_unknown.strip('[')
        missing_or_unknown = missing_or_unknown.strip(']')
        missing_or_unknown = missing_or_unknown.split(sep=',')
        missing_or_unknown = [int(value) if (value!='X' and value!='XX' and value!='') else value for value in missing_or_unknown]
        if missing_or_unknown != ['']:
            clean_df = df.replace({feat_info.iloc[x]['attribute']: missing_or_unknown}, np.nan)
        
    for i in df.columns:
        clean_df = clean_df.replace({i: ['XX', 'X']}, np.nan)
    
    # drop the outlier columns from the dataset (Columns with mean > 0.3)
    drop_columns = ['AGER_TYP','GEBURTSJAHR','TITEL_KZ','ALTER_HH','KK_KUNDENTYP','KBA05_BAUMAX']
    clean_df = clean_df.drop(drop_columns, axis=1)
    
    
    drop_multi = ['CJT_GESAMTTYP', 'FINANZTYP', 'GFK_URLAUBERTYP', 'LP_FAMILIE_FEIN', 'LP_FAMILIE_GROB', 'LP_STATUS_FEIN', 'LP_STATUS_GROB',
                  'NATIONALITAET_KZ', 'SHOPPER_TYP', 'ZABEOTYP', 'GEBAEUDETYP', 'CAMEO_DEUG_2015', 'CAMEO_DEU_2015']
    clean_df = clean_df.drop(drop_multi, axis=1)

    
    # select, re-encode, and engineer column values.
    clean_df['ANREDE_KZ'].replace([2,1], [1,0], inplace=True)
    clean_df['VERS_TYP'].replace([2.0,1.0], [1,0], inplace=True)
    clean_df['OST_WEST_KZ'].replace(['W','O'], [1,0], inplace=True)
        
    
    #engineer mixed-type features
    clean_df['DECADE'] = clean_df['PRAEGENDE_JUGENDJAHRE']
    clean_df['MOVEMENT'] = clean_df['PRAEGENDE_JUGENDJAHRE']
    
    decade_dict = {1:1, 2:1, 3:2, 4:2, 5:3, 6:3, 7:3, 8:4, 9:4, 10:5, 11:5, 12:5, 13:5, 14:6, 15:6}
    movement_dict = {1:1, 2:0, 3:1, 4:0, 5:1, 6:0, 7:0, 8:1, 9:0, 10:1, 11:0, 12:1, 13:0, 14:1, 15:0}

def get_kmeans_score(data, center):
    '''
    returns the kmeans score regarding SSE for points to centers
    INPUT:
        data - the dataset you want to fit kmeans to
        center - the number of centers you want (the k value)
    OUTPUT:
        score - the SSE score for the kmeans model fit to the data
    '''
    #instantiate kmeans
    kmeans = KMeans(n_clusters=center)

    # Then fit the model to your data using the fit method
    model = kmeans.fit(data)
    
    # Obtain a score related to the model fit
    score = np.abs(model.score(data)) #absolute value taken
    
    return score
